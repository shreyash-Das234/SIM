4. To create a wordlist that uses the character set  - vesit@123
> crunch 5 7 -o wordlist.txt -c vesit@123 -b 3mb
-  5  7: Generates words of length between 5 and 7 characters
-  -o START: Output file will be named START.
-  -c flag for specifying the custom charset directly
- -b 3mb: This option limits the output to 3 MB in size

to uncompress file: gunzip wordlist.txt.gz
